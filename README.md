# Spotly 🔍

Spotly is a simple offline application that helps you remember where you placed your belongings.

## Features

- Save item names and locations.
- Organize items by category.
- Instant search.
- Local storage (LocalStorage).
- Works completely offline.
- No ads.
- No account required.
- No permissions required.

## Privacy

Spotly does not collect, transmit, or share any user data.

All information is stored locally on the user's device.

## Technologies

- HTML5
- CSS3
- JavaScript
- LocalStorage

## Installation

1. Download or clone the project.
2. Open `index.html` in a browser.

Or build the Android APK using your preferred WebView tool.
